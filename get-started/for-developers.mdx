## Authentication

Send an HTTP header:

```text
Authorization: Bearer <API_KEY>
```

## Sandbox

Use `https://sandbox.api.logivice.com` to test without affecting production data.

## SDKs

| Language | Package |
|----------|---------|
| PHP      | `composer require logivice/sdk` |
| JS/TS    | `npm install @logivice/sdk` |
